# Satyam Saxena Portfolio

Portfolio site published with GitHub Pages at:

https://satyamsaxena1.github.io/

The site presents applied AI, railway inspection, GPS tooling, and public project work through an interactive, illustrated tiny-world experience. It is built with plain HTML, CSS, canvas, and JavaScript for direct GitHub Pages deployment.

The railway project panel lazy-loads an interactive MapLibre route layer using extracted GPS points from the published Folium railway map. It runs keyless with a public OpenStreetMap basemap by default; an optional `window.ARCGIS_API_KEY` value can be added later to use ArcGIS Location Platform basemap styles.
